Detailed Description

template<typename Source>
struct kwk::view_span< Source >

SCARY base class gathering all types and member functions pertaining to the upkeep, access and update to the span-like part of a kwk::view.

Contrary to most standard view and container,kwk::view_span keep the potential const qualifier of underlying type on purpose.
