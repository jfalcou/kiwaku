Detailed Description

template<std::ptrdiff_t Size, std::size_t Alignment>
struct kwk::shallow_allocator< Size, Alignment >

Model of the kwk::concepts::allocator concept. kwk::shallow_allocator allocates and deallocates memory block using a pre-existing, externally owned memory block source.

SizeTotal capacity a kwk::shallow_allocator can allocate during its lifetime
