Constructs from a shape with a higher order.

Constructs a kwk::shape from a shape with higher orders, compacting the extra-dimensions into the last.

If you require compile-time shape descriptor to be updated, consider using compress.

This constructor does not participate in overload resolution of the shape is not fully specified at runtime.

{
// x will contains 160 x 2
kwk::shape< kwk::_2D > x(kwk::of_size(4, 8, 5, 2));
std::cout << x << "\n";
// y will contains 1 x 1 x 7
kwk::shape< kwk::_3D > y(kwk::of_size(7));
std::cout << y << "\n";
}
